JSF: Add Conversation ID to a link or button

January 17, 2015
JSF gotcha

To add the conversation ID parameter cid to a h:link or h:button:

<h:button value="Go" outcome="go">  
    <f:param name="cid" value="#{javax.enterprise.context.conversation.id}" />
</h:button>
